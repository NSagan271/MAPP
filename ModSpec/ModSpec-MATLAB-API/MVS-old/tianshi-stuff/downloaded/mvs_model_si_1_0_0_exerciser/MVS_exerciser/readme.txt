daa_mosfet.m: MVS model file (Check model documentation for details).

model_exerciser.m: Plots current, charges, and capacitances for both 32 nm and 45 nm data sets using the optimized set of parameters.

Optimized parameters are stored in files called "coeff_op_final_32nm.txt" and "coeff_op_final_45nm.txt"
Format is: Rs0, Rd0, delta, n0, nd, vxo, mu, Vt0 
These optimized parameters are obtained using "MVS Model parameter extractor".
